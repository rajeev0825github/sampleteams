# Playerize
### About 
A sample project for managing sports teams.

### Framworks used

Laravel and Vue.js

### Backend Testing
```sh
$ php artisan test --env=testing
```
> Make sure to create .env.testing on your environment.

## Bundling assets
```sh
$ npm run watch
or
$ npm run dev
```
